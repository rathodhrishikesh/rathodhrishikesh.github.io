FIX Protocol Data Extractor

Authored By: Hrishikesh Rathod

Introduction

  • This mini project revolves around efficiently extracting specific tag values from log files in FIX (Financial Information eXchange) protocol format using Python and regular expressions.
  • To ensure compatibility with Excel and maintain data integrity, a custom sanitization function is implemented.
  • The extracted and sanitized data is then exported to an Excel file, facilitating organized and insightful data analysis.

Technical Skills Used

  • Python
  • Excel

Results

  • Extracted specific tag values from the FIX protocol log file.
  • Handled issues related to illegal characters in tag values and Excel compatibility.
  • Stored the extracted data in an Excel spreadsheet for further analysis.

Concepts Learned

  • FIX protocol logs analysis
  • Regular expressions and pattern matching techniques
  • Cross platform compatibility between Excel and Python
  • Pandas DataFrames: Structured data storage and manipulation
  • Error handling

Methodology

  • Data Extraction: The Python script reads a log file in the FIX protocol format, iterating through each line. It uses regular expressions to extract specified tag values (such as 35, 49, 56, 117, and 6001) from each line.
  • Data Sanitization: To ensure compatibility with Excel, a custom function was implemented to sanitize extracted values. This function removes characters not allowed in Excel sheet names, ensuring data integrity.
  • Data Storage: Extracted and sanitized tag values are stored in a Pandas DataFrame, which provides a convenient data structure for manipulation.
  • Data Export: The DataFrame is exported to an Excel file using the Pandas to_excel function, creating a structured dataset that can be easily shared and analyzed.

Project Outcome

  • The "FIX Protocol Data Extractor" tool successfully demonstrated the extraction of specific tag values from a FIX protocol log file and their storage in an Excel spreadsheet. By addressing compatibility issues and using appropriate data structures and libraries, the project provides a foundation for further analysis, reporting, and visualization of financial data.
  • Overall, this mini-project showcased the practical application of programming skills to process financial data and generate insights in a real-world context.
